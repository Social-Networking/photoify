# Photoify

Instagram clone in Laravel

## Installation

### Requirements

- PHP
- Node & NPM
- Composer

### Install

1. Clone the repo: `git clone https://github.com/laykith/photoify.git`
2. Open the folder `cd photoify`
3. Run `./install.sh`

#### Optional

You can run `php artisan db:seed` to seed the database with demo data
